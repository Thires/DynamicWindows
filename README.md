# DynamicWindows
creates windows for DragonRealms when using genie frontend<br>
can handle:<br>
bank debt<br>
bank deposit<br>
bank transfer<br>
bank withdraw<br>
bug window<br>
injury window for self<br>
injury window for others<br>
profile /edit<br>
store window<br>
tdp plan<br>
spells choosing<br>
magical feats choosing<br>
